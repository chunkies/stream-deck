MacroPad

Privacy Policy

Last updated: April 30, 2026

Short version: MacroPad collects nothing. It runs entirely on your local network. No accounts, no telemetry, no cloud.

What we collect

Nothing. MacroPad does not collect, transmit, or store any personal data, usage data, or telemetry.

The app runs a local HTTPS server on your own machine. Your phone connects to it over your local WiFi network. No traffic leaves your network except for the following optional features you explicitly configure:

  • Spotify integration — uses the local MPRIS interface; no Spotify account needed, no data sent externally
  • OBS plugin — connects to obs-websocket on your local machine only
  • Webhook sender action — if you configure a button to call a URL, that request goes to the URL you specified
  • Plugin marketplace — fetches a registry JSON from GitHub to list available plugins; no identity is sent

License validation (Pro)

When you enter a Pro license key, MacroPad validates it using an HMAC signature locally on your machine. No network request is made. The key itself is stored only in your local app config.

Local config storage

Your button layout, page config, and settings are stored in a JSON file in your OS app data folder:

  • Linux: ~/.config/macropad/config.json
  • macOS: ~/Library/Application Support/macropad/config.json
  • Windows: %APPDATA%\macropad\config.json

This file never leaves your machine unless you manually copy it.

Analytics on this website

This landing page uses Plausible Analytics — a privacy-friendly, cookieless analytics tool. Plausible does not use cookies, does not collect personal data, and is GDPR compliant. No IP addresses are stored. Page view counts help us understand how many people are interested in MacroPad.

Third-party services

MacroPad is hosted on GitHub Pages. Downloads are served from GitHub Releases. GitHub's own privacy policy applies to those interactions.

If you purchase Pro via Gumroad, Gumroad's privacy policy applies to that transaction. MacroPad itself never receives your payment details.

Children's privacy

MacroPad does not collect any information from anyone, including children.

Changes to this policy

If we add features that collect data, we will update this page and note the change date prominently.
